# Implementation Plan

- [x] 1. イベント検索フックの実装
- [x] 1.1 (P) useEventSearchフックのコアロジックを実装する
  - イベント配列を受け取り、検索クエリに基づいてタイトルの部分一致フィルタリングを行う
  - 大文字小文字を区別しない検索を実装する
  - 300msのデバウンスを適用し、入力ごとのフィルタリング実行を最適化する
  - 空文字列・空白のみの入力は検索未適用として扱い、元のイベント配列をそのまま返却する
  - 祝日イベントは検索対象外とする
  - 一致件数と検索適用状態を返却する
  - _Requirements: 1.2, 1.3, 1.5, 3.2, 3.3_

- [x] 1.2 (P) useEventSearchフックのテストを作成する
  - 部分一致フィルタリングの正確性（大文字小文字不区別）を検証する
  - 300msデバウンスの動作を検証する
  - 空クエリで全イベントが返却されることを検証する
  - 祝日イベントがフィルタ対象から除外されることを検証する
  - 検索クリア時に即座に全イベントが返却されることを検証する
  - _Requirements: 1.2, 1.3, 1.5, 3.3_
